Online Book Store Performance Analysis Using SQL & Power BI
📌 **Project Overview:**

This project analyzes online bookstore sales data using SQL for data extraction and analysis, and Power BI for interactive dashboard visualization. The objective is to identify sales trends, customer preferences, genre performance, and business insights that can support data-driven decision-making.

🎯 **Objectives:**
Analyze book sales performance across different genres.
Identify top-selling authors and books.
Track monthly sales trends.
Evaluate quantity sold across countries.
Create an interactive Power BI dashboard for business reporting.

🛠️ **Tools & Technologies:**
SQL
Power BI
Data Analysis
Data Visualization

📊 **Dashboard Features:**

Sales Analysis by Genre
Quantity Analysis by Genre
Monthly Sales Trends
Country-wise Quantity Distribution
Top Authors Performance
Year-wise Filtering (2022, 2023, 2024)

🔍 **Key Insights:**

Romance and Mystery genres generated the highest revenue.
Monthly sales exhibited seasonal fluctuations.
Certain countries consistently contributed higher order volumes.
Fiction authors accounted for a significant share of total sales.
Genre-wise analysis revealed clear customer reading preferences.

📈 **Analysis Performed Using SQL:**

Data Filtering and Sorting
Aggregation Functions (SUM, COUNT, AVG)
GROUP BY Analysis
ORDER BY Operations
Sales Trend Analysis
Genre Performance Analysis
Author Performance Evaluation

📷 **Dashboard Preview:**
Interactive Power BI dashboard created to visualize key bookstore performance metrics and support business intelligence reporting.

🚀 **Business Value:**
This project demonstrates how SQL and Power BI can be used together to transform raw sales data into actionable insights, helping businesses monitor performance, understand customer preferences, and improve decision-making.
